mac

  • Alfred 3 for Mac  https://www.alfredapp.com/

    Mac上最好用的效率启动工具。类似于windows下everything+wox的组合。强推！！！

  • snap  https://itunes.apple.com/cn/app/id418073146

    Mac平台下的快捷键设置App-在Mac下使用机械键盘！可以像windows下win+数字键打开快捷启动栏软件一样打开docker程序坞中的软件，并且可以配置软件的打开快捷键。强推！！！

  • iTerm2  https://www.iterm2.com/

    Mac上最高效的终端工具

  • iHosts  https://itunes.apple.com/cn/app/ihosts-%E7%BC%96%E8%BE%91%E7%A5%9E%E5%99%A8/id1102004240

    /etc/hosts 编辑神器，使用分组管理Hosts，灵活组合、一键切换。开发神器。

  • Vanilla  https://matthewpalmer.net/vanilla/

    状态栏图标折叠工具，避免多余的打扰

  • 腾讯柠檬清理  https://mac.gj.qq.com/

    腾讯出的一款免费mac清理工具，如果只是想要进行垃圾清理，这款完全满足需求

  • Paste  https://pasteapp.me/

    剪切板增强工具，现有最符合mac设计风格，使用体验最好的剪贴板增强工具

全平台

  • libreoffice  https://www.libreoffice.org/

    开源的office套件，可作为Microsoft及wps的补充

  • Visual Studio Code  https://code.visualstudio.com/

    全平台代码编辑工具，可安装插件进行markdown编写，或轻量级的前端代码编写，即开即用。之前markdown使用 typora 编辑，后来发现这个工具对于一些样式的处理还是不灵活，用 vscode 或者 atom 更加方便。

  • snipaste  https://zh.snipaste.com/

    是一个简单但强大的截图工具，也可以让你将截图贴回到屏幕上。如果你的大部分工作时间都在电脑前，贴图功能将改变你的工作方式、提升工作效率。暂时只有windows和mac版的，linux版本进行中。强推！！！

  • dbeaver  https://dbeaver.io/

    是一个通用的数据库管理工具和 SQL 客户端，支持 MySQL, PostgreSQL, Oracle, DB2, MSSQL, Sybase, Mimer, HSQLDB, Derby, 以及其他兼容 JDBC 的数据库。DBeaver 提供一个图形界面用来查看数据库结构、执行 SQL 查询和脚本，浏览和导出数据，处理 BLOB/CLOB 数据，修改数据库结构等等。全平台支持。

windows

  • everything  https://www.voidtools.com/

    文件全局检索工具。可以非常快速地检索到系统中你想找的文件。文件不知道放哪了？用这些工具找找看吧。强推！！！

  • ditto  https://ditto-cp.sourceforge.io/

    可以帮助你复制、粘贴多条记录，可设置剪贴板内容保留的个数与时间，非常方便。

  • Geek Uninstaller  https://geekuninstaller.com/

    专注于软件卸载

  • PotPlayer  https://potplayer.daum.net/?lang=zh_CN

    非常好使的视频播放器

  • QuickLook  https://github.com/QL-Win/QuickLook

    模拟mac中空格查看文件简略信息

  • MobaXterm  https://mobaxterm.mobatek.net/

    windows上全能的远程终端登录工具，个人版免费。

  • RedisDesktopManager  https://github.com/lework/RedisDesktopManager-Windows

    可下载安装包
